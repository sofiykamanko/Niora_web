# Niora_web
This repository contains the source code for the **NIORA** skincare website, a premium skincare brand that merges the power of nature and science. The website features an elegant, user-friendly design showcasing our product range, special offers, and company philosophy.

## Features

- **Shop Section**: Displays various skincare products such as serums, masks, and cleansers.
- **Floral Special**: Includes special offers, such as free gifts with purchases over a certain amount.
- **About Page**: Provides an overview of the brand's philosophy and approach to combining botanical ingredients with advanced skincare technologies.
- **Responsive Design**: Fully responsive layout optimized for both desktop and mobile devices.

## Technologies Used

- **HTML**: Markup language for the structure of the website.
- **CSS**: Styling and layout using custom styles for responsive design.
- **GitHub Pages**: The website is hosted on GitHub Pages for fast and easy hosting.
